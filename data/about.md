## Projects Overview

Haris has completed the following projects:

A content-based recommendation system that suggests similar movies using cosine similarity. It also fetches movie posters using the TMDB API.

### 2. Car Detection and Counting (YOLOv8 + ByteTrack)
A computer vision system that detects, tracks, and counts vehicles in video streams using YOLOv8 and ByteTrack.

### 3. Brain Tumor Detection and Segmentation
A medical imaging project that detects brain tumors using YOLO and performs segmentation using the Segment Anything Model (SAM).

### 4. Hand Sign Detection and Recognition
A real-time computer vision system that detects and recognizes hand gestures using a custom-trained dataset.

### 5. People Instance Segmentation
A project that performs instance segmentation on people using YOLO trained on a custom dataset.

### 6. LLM Fine-Tuning Notebooks
A set of experiments focused on fine-tuning large language models such as Gemma and LLaMA using custom datasets.

### 7. FastAPI Projects
A collection of backend API projects built using FastAPI to understand real-world API development.

### 8. LangChain Learning Repository
A practice-based repository exploring LangChain components such as chains, retrievers, document loaders, and vector databases.
