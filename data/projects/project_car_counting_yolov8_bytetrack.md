# Project: Car Detection, Tracking & Counting (YOLOv8 + ByteTrack)

## What it is
A pipeline that detects, tracks, and counts cars in highway video using YOLOv8 + ByteTrack with line-based counting.

## How it works
- Detect cars per frame (YOLOv8)
- Track IDs (ByteTrack)
- Count when vehicles cross a virtual line
- Export output video with overlays

## Tech / Tools
- Python, YOLOv8 (Ultralytics), ByteTrack, Supervision, OpenCV
