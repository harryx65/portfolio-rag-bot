# Project: Brain Tumor Detection & Segmentation (YOLO + SAM)

## What it is
A medical imaging project that detects brain tumors using YOLO and segments tumor regions using SAM (Segment Anything Model) for precise segmentation.

## Key features
- YOLO-based tumor detection
- SAM-based tumor segmentation for clear region highlighting
