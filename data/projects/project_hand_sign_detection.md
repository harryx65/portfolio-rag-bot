# Project: Hand Sign Detection & Recognition

## What it is
A custom-trained computer vision model for real-time hand detection and hand sign recognition using a custom dataset.

## What it includes
- Real-time detection + gesture classification
- Works on webcam and video input
- Designed to be extendable with new signs
