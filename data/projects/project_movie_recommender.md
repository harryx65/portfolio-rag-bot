# Project: Movie Recommendation System

## What it is
A Streamlit web app that recommends similar movies using cosine similarity and shows posters via the TMDB API.

## How it works
- Converts movies into vectors (feature engineering / NLP style)
- Uses cosine similarity to find nearest movies
- Returns top 5 recommendations
- Fetches posters from TMDB API
